## Free Astro Themes to Build a Blog

### AstroWind

<Picture src={img1} widths={[200, 400, 900]} sizes="(max-width: 900px) 100vw, 900px" alt="Astro Wind" />
AstroWind is a free and open-source template to make your website using Astro 2.0 + Tailwind CSS. It is designed taking into
account web best practices and supports Dark mode

<Button link="https://astrowind.vercel.app/" text="Demo" />

### Bookworm Light Astro

<Picture src={img2} widths={[200, 400, 900]} sizes="(max-width: 900px) 100vw, 900px" alt="Bookworm Light Astro" />

Bookworm Light is a feature-rich, minimal, highly customizable, easy-to-use free Astro blog theme. It is a minimal multi-author Astro blog theme that is perfect for any kind of blog website

<Button link="https://bookworm-light-astro.vercel.app/" text="Demo" />

### Blogster

<Picture src={img3} widths={[200, 400, 900]} sizes="(max-width: 900px) 100vw, 900px" alt="Blogster" />

This is a starter template for creating a blog with Blogster. It comes in four themes, including Bubblegum which is colorful and built with Tailwind. The template is fast, mobile-friendly, accessible, and has a perfect lighthouse score. It also offers extended markdown with custom components and a built-in RSS feed for SEO.

<Button link="https://blogster-bubblegum.netlify.app/" text="Demo" />

### Stablo Blog Template

<Picture src={img3} widths={[200, 400, 900]} sizes="(max-width: 900px) 100vw, 900px" alt="Stablo Blog Template" />
Stablo is a minimal and easy-to-use blog template that is free and open-source. It is built using Astro, Tailwind CSS, and
MDX, and is fully mobile-responsive with a clean and modern design. It offers configurable dark and light themes, About and
Archive pages, and a working contact page powered by Web3Forms. The template is sponsored by Web3Templates and can be launched
quickly for personal or startup blogs.

<Button link="https://stablo-astro.web3templates.com/" text="Demo" />
